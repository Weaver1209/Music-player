# local music player 
i built this to see if i could make a fully functional music player that runs entirely in the browser using strictly local files. no servers, no databases, no tracking. just pure html, css, and vanilla javascript.

## what it does

* **plays music**  
* **the bare necessities:** has a working seek slider, play/pause, next/prev, loop, and a shuffle button that actually works.

## what I will try to implement
* **local storage**
* **playlist ui**
